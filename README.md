# BOLC 🧠⚡️

A fast-paced, mental reflex game built to test your concentration and math skills under pressure. Playable on both desktop and mobile devices.

## 🎮 Play the Game
**https://hidmar.github.io/Bolc-game/**

## 📖 How to Play
The game counts up sequentially starting from 1. Your goal is to enter the next number in the sequence before the timer runs out. 

**The Catch:** If the upcoming number fits any of the "BOLC" rules below, you must press the **BOLC** button instead of typing the number:
* It is a multiple of 3 (e.g., 6, 9, 12)
* It is a multiple of 7 (e.g., 14, 21, 28)
* It contains the digit 3 (e.g., 13, 23, 31)
* It contains the digit 7 (e.g., 17, 27, 70)

## 🌟 Features
* **Three Difficulty Tiers:**
  * **Easy (15s):** The upcoming number is fully visible, and "danger" digits (3 and 7) are highlighted in red to train your brain.
  * **Medium (10s):** The upcoming number is hidden. You must rely entirely on your working memory to keep track of the sequence.
  * **Hard (5s):** The sequence is hidden, the timer is brutal, and a dynamic "Heat" system shifts the background from blue to deep red as time runs out, adding psychological pressure.
* **Global Leaderboard:** Integrated backend to track the Top 5 highest scores in the world.
* **Mobile-Optimized:** Features a custom, on-screen numpad with instant auto-submit logic so players can play seamlessly with just their thumbs.
* **Keyboard Support:** Desktop players can type numbers normally and hit the `Spacebar` to trigger a BOLC for lightning-fast gameplay.
